# Security policy

## Supported versions

Only the latest commit on `main` receives security fixes. This is a
personal homelab project with no formal release cadence — see the
README disclaimer.

## Reporting a vulnerability

Please use GitHub's **private vulnerability reporting** for this repo:

https://github.com/bradydibble/llm-relay/security/advisories/new

If that interface is unavailable, you can email the maintainer directly
(see the commit history for the address). Please do **not** open a
public issue for security-sensitive reports.

You can expect:

- An acknowledgement within 7 days
- A fix or a documented "won't fix" decision within 30 days for issues
  I can reproduce
- Credit in the fix commit and any release notes, if you'd like it
